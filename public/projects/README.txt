HOW TO ADD YOUR PROJECT SCREENSHOTS
=====================================

Drop your screenshots into this folder with these exact filenames:

  screenshot-1.jpg  →  Smart Parking System
  screenshot-2.jpg  →  Tomato Disease Detection
  screenshot-3.jpg  →  GoClinic Healthcare

Supported formats: .jpg, .jpeg, .png, .webp

Recommended size: 600×340 pixels (or any 16:9 ratio)

Once you add the images, they will automatically appear in the
Projects section when someone clicks the Projects shortcut.

Until you add them, a gradient placeholder is shown instead.
